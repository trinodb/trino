# Lane Audit Template

Use this template shape when refining or adding lane-level audit content beyond the generated skeleton.

## Required Sections

- Lane summary
- Legacy suites retired in this lane
- JUnit suites introduced in this lane
- Existing suites extended in this lane (if any)
- Environments introduced in this lane
- Ported test classes
- Existing test classes extended for this lane (if any)
- Lane-level open questions / intentional differences
- Per-class sections
- Per-method sections

## Per-Method Checklist

- Legacy source method
- Current target method
- Mapping type
- Environment parity
- Tag parity
- Setup parity
- Action parity
- Assertion parity
- Cleanup parity
- Any observed difference, however small
- Known intentional difference
- Reviewer note
- Audit status
