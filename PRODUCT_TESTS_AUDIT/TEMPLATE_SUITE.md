# Suite Audit Template

Use this template shape when refining or adding suite-level audit content beyond the generated skeleton.

## Required Sections

- Suite summary
- Environment runs
- CI wiring
- Parity checklist

## Per-Suite Checklist

- Legacy suite or lane source
- Current suite class
- Explicit runs and environments
- Include tags
- Exclude tags
- Expected migrated class count
- Expected migrated method count
- Expected migrated classes covered
- Expected migrated methods covered
- CI bucket in `.github/workflows/ci.yml`
- Special secret/credential gate
- Legacy launcher suite removed
- Parity status
